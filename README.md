4月18日 小组第一次讨论
项目实施大体思路：由重力加速度传感器（姿态模块）判断摇的动作，将反馈数据传递给单片机，单片机通过协议将数据传递给LCD屏，实现计数。设置三个按键：RESET清零键，开始键，排序键。
项目分工：分为两组，一组两人（中期实施方案+三个按键的设计），另一组三人（负责姿态模块与工作日志）
